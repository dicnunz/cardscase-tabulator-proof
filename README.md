# C.A.R.D.S. Tabulator Proof

Standalone implementation for the public $1,000 C.A.R.D.S. Tabulator gig.

## Scope

- Vanilla JS and Tabulator CDN.
- Synthetic NBA card JSON only.
- Sortable value, grade, and acquired-date columns.
- Global player search.
- 20-row pagination.
- Round image avatars, bold titles, grade badges, currency formatting, and status tags.
- Dark navy dashboard styling with orange/gold accents.

## Local Run

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.
